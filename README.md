<h3># LinkedIn clone </h3>
<p>LinkedIn is a business and employment-focused social media platform that works through websites and mobile apps. It is now owned by Microsoft. The platform is primarily used for professional networking and career development, and allows jobseekers to post their CVs and employers to post jobs.</p>

<h3># Getting Started</h3>
<p>Prerequisites : React</p>

<h3># Installing</h3>
<p>Clone the repository
git clone https://github.com/jaideep0101/linkedin-clone.git
cd linkedin-clone</p>

<h3># Install dependencies</h3>
<p>npm install</p>

<h3># Create a Firebase project and add your Firebase config object to src/firebase.js </h3>
<p>const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  databaseURL: "YOUR_DATABASE_URL",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID",
  measurementId: "YOUR_MEASUREMENT_ID",
};</p>

<h3># Enable the Firebase Authentication and Firestore services in your Firebase project</h3>
<p>#Start the development server</p>
<p>npm start</p>

<p>Navigate to http://localhost:3000 in your web browser</p>
